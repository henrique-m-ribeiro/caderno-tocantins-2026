# Amostra de PDFs para Análise de Viabilidade (Fase 1)

## Objetivo

Este diretório contém uma amostra estratificada de 10-15 PDFs dos Perfis Socioeconômicos Municipais da SEPLAN-TO para análise detalhada de viabilidade antes da extração em massa.

## Estratificação da Amostra

Conforme recomendação Manus (Framework IA-Collab-OS):

### Municípios Grandes (3-4 PDFs) - População > 50.000
- Palmas (capital, ~313 mil habitantes)
- Araguaína (~180 mil)
- Gurupi (~87 mil)
- Porto Nacional (~53 mil)

### Municípios Médios (4-5 PDFs) - População 10.000-50.000
- Representando diferentes regiões de planejamento
- Diferentes portes dentro da faixa

### Municípios Pequenos (3-4 PDFs) - População < 10.000
- Representando diferentes regiões de planejamento
- Incluindo o menor município (Oliveira de Fátima, ~1.100 hab)

## Cobertura Regional

A amostra deve cobrir as **8 Regiões de Planejamento SEPLAN-TO 2024**:
1. Bico do Papagaio
2. Norte
3. Meio Norte
4. Vale do Araguaia
5. Central
6. Jalapão
7. Sul
8. Sudeste

## Critérios de Seleção

- **Diversidade de porte:** Pequeno, médio, grande
- **Diversidade regional:** Todas as 8 regiões representadas
- **Diversidade econômica:** Agro, indústria, serviços
- **Casos especiais:** Capitais regionais, municípios fronteiriços, etc.

## Análise a Ser Realizada

Para cada PDF da amostra, documentar:
1. **Estrutura geral:**
   - Número de páginas
   - Organização dos capítulos
   - Presença de índice/sumário
2. **Qualidade técnica:**
   - PDF nativo vs escaneado
   - Necessidade de OCR
   - Qualidade de tabelas
3. **Indicadores presentes:**
   - Quais dos nossos indicadores estão presentes
   - Localização (capítulo, página)
   - Formato (tabela, texto, gráfico)
4. **Variações identificadas:**
   - Diferenças de layout entre municípios
   - Nomenclaturas variadas
   - Indicadores ausentes
5. **Estratégia de extração:**
   - Padrões de extração identificados
   - Exceções que requerem tratamento especial
   - Estimativa de taxa de sucesso
